5.17 问题

  1. 怎么给mysql 唯一key 取名

    答: index,key在MySQL指代的是同一个东西，不过在key有一个unique key 它会限制多行的该列不会有相同值。

可以看到我们自定义的unique key 名为idx_language_code_flag,可以用show index from language_tbl;查看一下表的index。另外如果表已经建立好:create index idx_language_code on language_tbl(language_code);，其中idx_langauge_code是自定义的名字。

  1. 删除表

    答:drop table if EXISTS language_tbl; 删除表,包括表数据。

    `delete from language_tbl;`删除表的所有已有数据。

  2. 当访问一个URL时报404，nginx/1.10.2，这种情况查看 nginx 日志判断看是否Nginx访问到了。
    答: 查看日志，第一步要确定我们要查看的那个日志在那个位置，进入到/etc/nginx目录使用grep，找到log目录。

5.15 问题

  1. Mysql 使用group by是否可以将多行中同列的不同值合并成一条

    答: 主要使用group_concat()函数，可以将同组id相同的值合并到一个值当中

  2. 一条SQL更新两个表

    答: update tbl_name1 a, tbl_name2 b set a.xx='',b.xx='';这样有个不好的地方。这两个表的所有字段都会更新。所以这种情况的使用场景是两个表有唯一主键关联，一定要仔细检查。

  3. redis Key 加密登录

    答:如果redis配置了密码，修改了redis.cnf 的requirepass chen；

5.13 问题

  1. mysql 替换数据库的某些值

    答案: 使用MySQL的replace()函数

  2. mysql 字符相加

    答: 使用MySQL的CONCAT(str1, str2, str3, ...)
